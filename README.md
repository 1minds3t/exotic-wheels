# exotic-wheels (CF Pages host)

This repo powers **[exotic-wheels.pages.dev](https://exotic-wheels.pages.dev/)** via Cloudflare Pages.

It is infrastructure only. For everything else:

| | |
|---|---|
| **Package index (pip)** | [exotic-wheels.github.io](https://exotic-wheels.github.io/) |
| **Issues / wheel requests** | [exotic-wheels/exotic-wheels.github.io/issues](https://github.com/exotic-wheels/exotic-wheels.github.io/issues) |
| **Releases / wheels** | [exotic-wheels/exotic-wheels.github.io/releases](https://github.com/exotic-wheels/exotic-wheels.github.io/releases) |
| **Discussions** | [exotic-wheels/exotic-wheels.github.io/discussions](https://github.com/exotic-wheels/exotic-wheels.github.io/discussions) |

---

`/simple/*` redirects to `exotic-wheels.github.io/simple/*` via `_redirects`.
