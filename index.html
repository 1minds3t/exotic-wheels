🛞 exotic-wheels

Pre-built Python wheels for exotic platforms that PyPI doesn't cover (musllinux armv7l, riscv64, s390x, etc.)

Install

pip install <package> --extra-index-url https://exotic-wheels.pages.dev/

Available packages

  • cryptography — musllinux armv7l
